# Innovation_station

### Setup
```bash
git clone https://github.com/8ball030/innovation_station
cd innovation_station
poetry install
```

### Testing
```bash
make test
```

### Linting
```bash
make lint
```

### Formatting
```bash
make fmt
```

### Releasing
```bash
make release
```
